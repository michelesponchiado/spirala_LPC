void memcpyram2flash(const void *dest,
void *src,
unsigned int memsize)

Copies contents of RAM variable into Flash program memory space.

Its use can be dangerous, since it may “reprogram” the microcontroller if wrong used.  It is tested to work with dsPIC30Fxxx.  Function takes about 10 ms, because writing to flash is slow.  If reset occurs between erasing and writing new values, up to 64 bytes of memory locations may stay erased.

Parameters

destDestination address in flash memory.
srcSource address in RAM memory.
memsizeNumber of bytes to copy.
